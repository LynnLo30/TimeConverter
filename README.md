# 🕒 Time Converter

[Live Demo](https://lynnlo30.github.io/TimeConverter/)

## 功能特色

- 🌍 **時區轉換**：快速掌握不同時區（紐約/倫敦）的時間
- ⏱️ **即時時鐘**：自動更新，隨時掌握正確時間
- 🖱️ **面板操作**：點擊按鈕即可移動時間面板
- 📋 **複製到剪貼簿**：一鍵複製轉換後的時間摘要

## 技術架構

- HTML / CSS (SCSS)
- 原生 JavaScript（純函式設計）
- GitHub Pages 部署

## 專案目的

此專案作為個人前端練習，目標是：

- 強化 **純函式架構** 的思維
- 學習 **GitHub flow 開發流程**
- 打造一個實用、即時、跨平台的工具
